## 🚀 Getting Started (Speed Run Any%)

### The Essentials

- Node.js 16.x or higher
- npm or yarn
- Git (obviously)

### Quick Start Guide

1. Grab the code:

```bash
git clone https://github.com/yourusername/aniheaven.git
```

2. Install the goods:

```bash
cd aniheaven
npm install
```

3. Let it cook:

```bash
npm run dev
```
